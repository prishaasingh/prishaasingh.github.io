# Who Votes for What? Understanding Working Class Voter Trends and Partisan Shifts


# Brief Introduction 
This project looks to assess the trends of the way the working class votes on policies that are produced by the Democratic Party, which would otherwise benefit working-class voters, to see if the trend exists and how large it is. This study also seeks to understand if region, race, and educational attainment have an impact on working voters, as well as certain social policy issues.

# Data
To complete this study, all data were taken from the Cooperative Election Study Recontact Survey. The data includes survey results from 2020 to 2024.
